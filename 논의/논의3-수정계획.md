# 논의3: 수정 계획

**날짜**: 2025-12-16
**목표**: 3가지 문제 모두 수정

---

## 📋 수정할 문제 리스트

### ✅ 1. 핫키 문제 (High Priority)
**문제**: 다른 파일로 이동해도 Vault Council이 자동 업데이트 안됨

**해결 방법**:
- `refreshContext()` 메서드 추가
- 핫키 실행 시 이미 열려있으면 컨텍스트 새로고침
- Workspace의 `active-leaf-change` 이벤트 리스닝

**예상 시간**: 10분

---

### ✅ 2. 번개 버튼 문제 (High Priority)
**문제**: 볼트를 바꾸고 싶은데 설정만 열림

**해결 방법**:
- Obsidian API에서 볼트 목록 가져오기
- Modal로 볼트 선택 UI 표시
- 선택하면 해당 볼트로 전환
- 또는 Command Palette 열어서 "Open another vault" 명령 실행

**예상 시간**: 20분

---

### ✅ 3. Custom Mode 구현 (Medium Priority)
**문제**: 여러 파일을 직접 선택해서 분석하는 기능 없음

**해결 방법**:
1. 파일 선택 버튼 추가
2. `SuggestModal`을 사용한 파일 검색/선택 UI
3. 선택된 파일 목록 표시
4. 선택된 파일들의 내용을 컨텍스트로 사용

**예상 시간**: 30분

---
